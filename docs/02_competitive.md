# 02 競合分析（きめのすけ）

作成日: 2026-07-08 / 最終改訂: 2026-07-11 / フェーズ: Phase 0（構想）

> **調整さん等の日程調整ツールは非競合として除外する。**
> 調整さん・伝助・Doodle 等は「日程（いつ）」を決めるツールであり、本サービスの対象は「行き先・対象（どこ/なに）を複数人で決める」意思決定支援である。ユーザー登録不要・共有URL方式という体験は参考（ベストプラクティスとして流用）にするが、解く課題が異なるため競合比較の対象からは外す。

**差別化の訴求主軸: 候補ごとの全員の意見と、人気・反対・安全な代替案を同時に見える化する。**
単純な多数決（○の数）では少数派の強い反対が無視され、後から追加された候補も不利になりやすい。きめのすけは未評価を区別し、Candidate作成時刻、×、❤️、🌀、コメントを候補カード内で比較し、clear / discussion / fallbackの3状態で議論材料を示す。

## 競合マップ

| 競合                           | 種別         | 登録要否           | 拒否権（×）の扱い          | 複数判断基準  | 収益モデル        | 主な弱点（きめのすけ視点）                       | 出典                                                                               |
| ---------------------------- | ---------- | -------------- | ------------------ | ------- | ------------ | ----------------------------------- | -------------------------------------------------------------------------------- |
| LINE 投票（トークの投票機能）            | チャット内アンケート | LINEアカウント必須    | なし（単純多数決/複数選択）     | なし      | LINE本体（間接）   | 反対意見・懸念を構造的に扱えない。LINE利用者/グループ内に閉じる  | [公式ガイド](https://guide.line.me/ja/chats-calls-notifications/chats/chat-poll.html) |
| Google マップ グループプランニング（共有リスト） | 地図ベース共同計画  | Googleアカウント必須  | なし（♡リアクション/コメントのみ） | なし（♡のみ） | Google本体（間接） | 賛否の集約・確定の概念がない。反対の可視化や保留の仕組みがない     | [マップ ヘルプ](https://support.google.com/maps/answer/7280933?hl=ja-JP)               |
| 多数決さん 系（多数決さん / かんたん投票 等）    | Web投票ツール   | 不要（多くは登録不要）    | なし（多数決）            | なし      | 広告           | 名前どおり「多数決」で決着。拒否権や懸念解消のフローがない       | [多数決さん](https://tasuketsu.com/)                                                  |
| 旅行計画アプリ（Wanderlog 等）         | 旅程共同編集     | アカウント必須（共同編集時） | なし（投票/賛否機能は主眼でない）  | なし      | フリーミアム（サブスク） | 「計画・旅程作成」が主眼で「複数人で1つに決める」意思決定支援ではない | [Wanderlog](https://wanderlog.com/)                                              |

## 観点別の比較サマリ

| 観点 | 既存競合の傾向 | きめのすけ |
|---|---|---|
| 参加のハードル | アカウント/アプリ必須が多い | 登録不要・共有URLで参加、ログインは任意 |
| 意思決定の方式 | 多数決（○の数）中心 | ○数と×有無からclear / discussion / fallbackを可視化 |
| 少数派の反対 | 無視される/可視化されない | ×と回答者を常時表示し、○最多でもdiscussionとして明示 |
| 懸念の解消 | 仕組みなし | ×→− への変更で解消を促す（誰でも変更可） |
| 評価の多面性 | 単一の賛否のみ | ○/−/× に加え ❤️（非決定ポジ）・🌀（非決定ネガ）・コメント |
| 対象領域 | 日程調整/計画/一般投票に分散 | 種別に縛られない「どこ/なにを選ぶか」の共同意思決定 |
