## 改訂履歴

- **2026-07-11 共同編集型・回答者行モデル（ADR-0006）**: トップと候補追加フォームからお名前欄を撤去。Event詳細上部に回答者セレクターを置き、候補カード内の選択行だけを編集可能にする。未評価と能動−を分離し、3種類の最終候補状態を追加。新規ラベル・色・アイコンのexact visualは実装後の人間確認で確定する。
- **2026-07-10 属性撤廃（ADR-0005）**: 属性選択・属性表示ラベル・属性連動お題placeholder・eyebrowの属性表示を撤去。お題placeholderは汎用。判断基準の実装は Slice 5。
- **2026-07-09 文言方針転換：漢字優先化。** 情報密度優先（スマホでの文字数増・表示崩れを回避）。ひらがな多め方針から、漢字を積極使用する方針へ。例外でひらがな維持＝ブランド「きめる」系（きめのすけ／きめよう！／きめの すけざえもん）。操作系ボタンは通例表記（例: キャンセル）。
- 2026-07-09 初版確定（きめのすけ口調・ひらがな多め）。Slice 1 で実装（`96c7622`）。

## トーン方針（2026-07-09 改訂）

- **漢字を積極使用**（情報密度優先・スマホ表示崩れ回避）。
- ひらがな維持の例外: ブランド「きめる」系（きめのすけ／きめよう！／きめの すけざえもん）。※旧「属性表示ラベル」は属性撤廃（ADR-0005）に伴い対象外。
- 操作系ボタンは通例表記に倣う（例: キャンセル）。
- 「きめたいひと」「オーナーメニュー」という語はUIで使わない。

## 重要な整合ルール（実装厳守）

> **⚠ ADR-0005（2026-07-10）で「属性」を撤廃。** 以下に残る属性関連（**属性選択・属性表示ラベル・属性連動のお題placeholder・イベント詳細 eyebrow の属性表示**）は**撤去対象**。お題placeholder は**汎用**にする。属性関連の記述は履歴として残すが、実際の撤去は Codex「属性最小撤去」タスクで行う。判断基準（Criterion）の実装は Slice 5。
> 漢字優先化改訂（下記）は属性に依存しない部分は有効。

## 回答者セレクター（ADR-0006）

- トップのお題作成画面とCandidate追加フォームにお名前欄を置かない。
- Event詳細のCandidate群より前に、選択中の回答者行と新しい名前入力を扱うセレクターを置く。
- 非選択回答者行は閲覧専用、選択行だけに○ / − / ×、❤️、🌀、コメントの編集controlを表示する。
- 「ゲスト」「本人認証」「参加登録」という語をユーザーへ見せない。
- 同名時は同じ人か確認し、別人なら異なる名前の再入力を求める。
- 回答者セレクター、同名確認、名前変更、回答者削除のexact copyは、コードベースワイヤーフレームと375px / 1366px実画面確認後に本書へ追記する。構造と挙動は[ADR-0006](../adr/0006-collaborative-response-row-model.md)および[実装仕様](collaborative-response-row-spec-draft-2026-07-11.md)を正とする。

### オーナー編集（同一画面方針・確定）

- **オーナーとゲストは同一レイアウト**。機能差は「お題・メモを編集できるのはオーナーだけ」の一点のみ。
- **「オーナーメニュー」独立見出し・パネルは廃止**。
- **所有表示（控えめ）**: 小さく「**あなたは お題とメモを直せます**」＋淡い色/小アイコン。
- **編集手段**: お題／メモの近くに小さな「**直す**」→ 編集 → 保存「**保存**」。
- 入力ラベルは「**お題**」「メモ」。

### Candidateカード・回答者行（ADR-0006）

- Candidate追加フォームにお名前欄を置かない。
- Candidateカードに全回答者行を表示し、未評価と能動−を区別する。
- Candidate作成時刻は「1時間以内に追加 / N時間前に追加 / N日前に追加」とする。クライアント時計より未来の場合は「1時間以内に追加」へclampする。
- 最終候補状態は`clear / discussion / fallback / none`のsemantic tokenを用い、色だけでなく状態ラベルまたはアイコン等を併用する。
- exact color、状態ラベル、アイコン、配置、追加時刻copyの最終調整は実画面確認後に確定する。

### コピーボタン（確定・全画面共通）

- 既定ラベル「コピー」。**成功時はチェックマーク（✓）に変化**（約1.8秒で戻る）。失敗時の文言は出さない。
