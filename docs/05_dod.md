## スライスDoD（全スライス共通）

- [ ] 受け入れ条件（Given/When/Then）を全て満たす
- [ ] スマホ幅（375px）とデスクトップ幅の両方で表示崩れなし（同格）
- [ ] 主要操作のE2Eテストが自動化されグリーン
- [ ] エラー時にユーザー向けメッセージ表示（白画面禁止）
- [ ] 実装が仕様（要件定義書・データモデル・ADR）を勝手に変えていない

---

## リリースDoD（MVP全体）

- [ ] 全スライスDoD達成
- [ ] QAシナリオ全件合格
- [ ] 本番でオーナー1名＋ゲスト2名の実データテスト完了（ログインなし・Cookieベース）
- [ ] オーナー編集URLでの権限回復が動作（Cookie消失シナリオ）
- [ ] 検索エンジン非インデックス（noindex＋robots）設定済み
- [ ] 広告枠が実装され広告ネットワーク審査を申請済み（実表示は審査通過後）
- [ ] 利用規約・プライバシーポリシー掲出（広告・トラッキング・Cookie/ゲストトークン明記）
- [ ] 計測（アクセス解析）が動作
